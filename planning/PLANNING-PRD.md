# 꿈틀(Dream-Frame) — 기획 상세 PRD

## 1. 기능 명세

### 1-1. AI 진로 큐레이터

| 항목 | 내용 |
|------|------|
| 입력 | 학생이 책에서 발견한 감명 깊은 문장 (텍스트 입력) |
| 처리 | AI가 문장에서 핵심 가치 키워드 추출 (예: 도전, 협력, 창의성, 끈기) |
| 출력 | 추출된 가치와 연결된 직업군 2~3개 + 해당 역량 설명 카드 |

**동작 흐름:**
1. 학생이 텍스트 입력창에 문장을 붙여넣거나 직접 입력
2. "분석하기" 버튼 클릭
3. AI가 가치 키워드를 태그 형태로 표시
4. 각 키워드와 연결된 직업군/역량이 카드 UI로 제시
5. 학생이 관심 있는 카드를 선택하면 '나의 진로 보드'에 저장

### 1-2. AI 독서 코치 (질문 생성기)

| 항목 | 내용 |
|------|------|
| 입력 | 학생이 입력한 문장 + 책 제목/장르 정보 |
| 처리 | AI가 해당 문장의 맥락을 분석하여 자기 탐색형 질문 생성 |
| 출력 | 맞춤형 질문 2~3개 (객관식 아닌 서술형 유도) |

**질문 유형 예시:**
- 가치 탐색: "이 주인공이 중요하게 여긴 것은 무엇이고, 나도 그것을 중요하게 여기나요?"
- 미래 대입: "이 주인공의 선택이 나의 미래라면, 나는 어떤 결정을 내릴까요?"
- 강점 발견: "이 장면에서 내가 잘할 수 있을 것 같은 역할은 무엇인가요?"

### 1-3. 라이브 디지털 워크시트

| 항목 | 내용 |
|------|------|
| 실시간 협업 보드 | 학생들의 작성 내용이 포스트잇 형태로 교실 전체 화면에 실시간 표시 |
| 멀티미디어 기록 | 텍스트, 사진 업로드, 유튜브 링크 첨부 지원 |
| 반응 기능 | 친구 포스트잇에 좋아요/응원 댓글 달기 |
| 교사 대시보드 | 학생별 참여도, 작성 현황 실시간 모니터링 |

**동작 흐름:**
1. 수업 시작 시 교사가 '세션 열기' → 고유 세션 코드 생성
2. 학생들이 세션 코드로 입장
3. 각자 작성한 내용이 실시간으로 공유 보드에 반영
4. 수업 종료 시 교사가 '세션 닫기' → 결과 자동 저장

### 1-4. 게이미피케이션 (꿈 포인트 & 레벨)

| 활동 | 포인트 |
|------|--------|
| 문장 입력 및 AI 분석 완료 | +10 꿈P |
| 질문에 대한 답변 작성 | +15 꿈P |
| 진로 한 문장 생성 | +20 꿈P |
| 친구에게 응원 댓글 달기 | +5 꿈P |
| 멀티미디어 자료 첨부 | +10 꿈P |

**레벨 체계:**
| 레벨 | 필요 포인트 | 칭호 |
|------|-----------|------|
| Lv.1 | 0 | 꿈 씨앗 |
| Lv.2 | 50 | 꿈 새싹 |
| Lv.3 | 150 | 꿈 탐험가 |
| Lv.4 | 300 | 꿈 설계사 |
| Lv.5 | 500 | 꿈 크리에이터 |

### 1-5. AI 비주얼 싱킹 (마인드맵 자동 생성)

| 항목 | 내용 |
|------|------|
| 입력 | 학생이 입력한 키워드 3~7개 |
| 처리 | AI가 키워드 간 관계를 분석하여 마인드맵 구조 생성 |
| 출력 | 시각적 마인드맵 이미지 (다운로드/공유 가능) |

**동작 흐름:**
1. 수업 활동 중 기록한 키워드를 자동 수집 또는 수동 입력
2. "마인드맵 만들기" 버튼 클릭
3. AI가 키워드를 중심 주제 / 하위 주제로 분류하여 마인드맵 생성
4. 학생이 색상/배치를 간단히 수정 가능
5. 완성된 마인드맵을 포트폴리오에 저장 또는 공유 보드에 게시

## 2. 사용자 시나리오

### 시나리오 1: 첫 수업 — "나의 가치 발견하기"

- **Who**: 중학교 1학년 김하늘 (자유학기제 진로탐색 수업)
- **When**: 자유학기제 첫 번째 진로 독서 수업 (45분)
- **Where**: 교실, 크롬북 사용
- **Why**: 책 속 인물의 가치관을 통해 자신의 가치관을 탐색하고 싶음
- **How**:
  1. 크롬북에서 꿈틀 접속 → 교사가 공유한 세션 코드 입력
  2. 오늘의 책 테마 확인: "도전과 실패"
  3. 책을 읽으며 인상 깊은 문장 입력: "실패해도 괜찮아, 다시 도전하는 게 중요해"
  4. AI가 분석: 가치 키워드 [도전, 회복력, 성장] → 관련 직업 [스타트업 창업가, 운동선수, 연구원]
  5. AI 코치 질문: "이 주인공처럼 실패 후 다시 도전한 경험이 있나요?"
  6. 답변 작성 후 '나만의 진로 한 문장' 생성: "나는 실패를 두려워하지 않고 도전하는 사람이 되고 싶다"
  7. 공유 보드에 게시 → 친구들 반응 확인 → 꿈 포인트 획득

### 시나리오 2: 교사 — "수업 운영과 모니터링"

- **Who**: 국어 담당 박선생님 (자유학기제 진로 독서 수업 운영)
- **When**: 수업 준비 및 수업 중
- **Where**: 교실, 교사용 PC/태블릿
- **Why**: 30명 학생의 활동을 실시간으로 파악하고 피드백하고 싶음
- **How**:
  1. 수업 전: 꿈틀에서 '새 세션 만들기' → 오늘의 책/테마 설정
  2. 수업 중: 대시보드에서 학생별 작성 현황 실시간 확인
  3. 참여가 저조한 학생에게 개별 알림 또는 직접 격려
  4. 수업 후: 학생별 활동 기록 PDF로 다운로드, 학부모 상담 자료로 활용

## 3. MVP 범위

### 필수 (MVP v1.0)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 문장 입력 & AI 키워드 분석 | 핵심 가치 추출 + 직업군 연결 | P0 |
| AI 질문 생성 | 자기 탐색형 질문 2~3개 제시 | P0 |
| 진로 한 문장 생성 | 활동 결과를 한 문장으로 정리 | P0 |
| 실시간 공유 보드 | 포스트잇 형태 실시간 공유 | P0 |
| 세션 관리 | 교사가 세션 코드로 수업 운영 | P0 |
| 반응형 웹 | 크롬북 브라우저에서 최적화 동작 | P0 |

### 추후 (v1.5~v2.0)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 게이미피케이션 | 꿈 포인트 & 레벨 시스템 | P1 |
| 멀티미디어 첨부 | 사진/유튜브 링크 업로드 | P1 |
| AI 마인드맵 생성 | 키워드 기반 비주얼 싱킹 | P1 |
| 교사 대시보드 | 학생별 참여도/기록 통계 | P1 |
| 좋아요/댓글 | 친구 포스트잇에 반응 | P2 |
| 포트폴리오 내보내기 | 학기별 활동 기록 PDF 출력 | P2 |
| 학부모 공유 | 활동 리포트 학부모 전달 | P3 |
