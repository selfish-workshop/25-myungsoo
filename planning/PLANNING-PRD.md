# 꿈틀(Dream-Frame) — 기획 상세 PRD

## 1. 서비스 구조

### 1-1. 사용자 역할

| 역할 | 설명 |
|------|------|
| 교사 | 세션 생성, 학생 계정 관리, 수업 운영, 대시보드 모니터링 |
| 학생 | 세션 참여, 독서 활동, AI 분석, 공유 보드 활동 |

### 1-2. 인증 방식

**교사**
- 별도 교사 계정으로 로그인

**학생**
- 교사가 학기 초 반별 학생 명단(이름, 번호) 일괄 등록
- 시스템이 반+번호 기반 로그인 정보 자동 생성 (예: 1반 3번 → `1-03`)
- 학생은 **반 선택 + 번호 입력 + 이름 확인**으로 로그인
- 규모: 8반 × 29~30명 = 약 240명

### 1-3. 화면 흐름

**학생 진입 흐름:**
```
로그인 (반+번호)
  → 홈 화면 (내 레벨 + 최근 활동 요약)
    → "수업 참여하기" 버튼
      → 세션 코드 입력
        → 수업 화면 (7단계 가이드)
```

**교사 진입 흐름:**
```
로그인 (교사 계정)
  → 교사 대시보드
    → "새 세션 만들기" 또는 "진행 중 세션 관리"
```

## 2. 수업 화면 흐름 (7단계 가이드형)

학생이 세션에 입장하면 단계별로 자동 전환되는 가이드 화면을 따릅니다.

| STEP | 화면 | 시간 | 설명 |
|------|------|------|------|
| 1 | 오늘의 테마 | 5분 | 오늘 읽을 책과 테마 확인 |
| 2 | 독서 & 문장 입력 | 10분 | 책 읽기 → 인상 깊은 문장 입력 → AI 키워드 분석 |
| 3 | AI 마인드맵 | 5분 | 추출된 키워드로 마인드맵 자동 생성 → 편집/저장 |
| 4 | 질문 & 답변 | 10분 | AI 코치 질문에 대한 내 생각 작성 |
| 5 | 진로 한 문장 | 5분 | 오늘 활동을 바탕으로 나만의 진로 문장 생성 |
| 6 | 공유 보드 | 7분 | 친구들 결과 보기, 좋아요/응원 댓글 (같은 반만 공개) |
| 7 | 결과 요약 | 3분 | 오늘 획득한 꿈 포인트, 내 기록 요약 |

**총 45분 수업 구성**

## 3. 기능 명세

### 3-1. AI 진로 큐레이터 (키워드 분석)

| 항목 | 내용 |
|------|------|
| 입력 | 학생이 책에서 발견한 감명 깊은 문장 (텍스트 입력) |
| 처리 | AI가 문장에서 핵심 가치 키워드 추출 (혼합형: 기본 목록 + AI 추가 제안) |
| 출력 | 추출된 가치와 연결된 직업군 2~3개 + 해당 역량 설명 카드 |

**키워드 체계 (혼합형):**
- 기본 가치 키워드 목록 (6개 카테고리, 약 30~40개) + AI가 추가 키워드 제안
- 교사가 키워드 허용/차단 설정 가능

| 카테고리 | 예시 키워드 |
|---------|-----------|
| 자기이해 | 자존감, 자신감, 정체성, 독립심 |
| 대인관계 | 협력, 소통, 배려, 리더십, 공감 |
| 성장가치 | 도전, 끈기, 성장, 회복력, 노력 |
| 사고역량 | 창의성, 비판적사고, 문제해결, 호기심 |
| 사회가치 | 정의, 책임감, 봉사, 환경, 다양성 |
| 진로태도 | 자기주도, 계획성, 적응력, 전문성 |

**직업군 연결 (혼합형):**
- 한국직업능력연구원 공식 직업 분류 체계 기반
- 학생 친화적 설명 함께 표시 (예: "데이터 분석가" → "숫자로 세상을 읽는 사람")

**동작 흐름:**
1. 학생이 텍스트 입력창에 문장을 붙여넣거나 직접 입력
2. "분석하기" 버튼 클릭
3. AI가 가치 키워드를 태그 형태로 표시
4. 각 키워드와 연결된 직업군/역량이 카드 UI로 제시
5. 학생이 관심 있는 카드를 선택하면 '나의 진로 보드'에 저장

### 3-2. AI 독서 코치 (질문 생성기)

| 항목 | 내용 |
|------|------|
| 입력 | 학생이 입력한 문장 + 책 제목/장르 정보 |
| 처리 | AI가 해당 문장의 맥락을 분석하여 자기 탐색형 질문 생성 |
| 출력 | 난이도에 맞는 질문 2개 |

**난이도별 질문 스타일 (교사가 세션 생성 시 설정):**

| 난이도 | 질문 스타일 | 예시 |
|--------|-----------|------|
| 쉬움 | 감정/호불호 중심, 짧게 답할 수 있음 | "이 장면을 읽으면서 어떤 기분이 들었나요?" |
| 보통 | 자기 경험과 연결, 2~3문장 서술 | "이 주인공처럼 실패 후 다시 도전한 경험이 있나요?" |
| 심화 | 가치 판단·미래 설계, 깊은 사고 유도 | "이 주인공의 가치관 중 내 진로에 가져가고 싶은 것은 무엇이고, 그 이유는?" |

- 모든 난이도에서 질문 **2개** 생성

### 3-3. AI 비주얼 싱킹 (마인드맵 자동 생성)

| 항목 | 내용 |
|------|------|
| 입력 | STEP 2에서 AI가 추출한 키워드 3~7개 |
| 처리 | AI가 중심-가지형 마인드맵 구조 생성 |
| 출력 | 시각적 마인드맵 이미지 (다운로드/공유 가능) |

**마인드맵 구조 (중심-가지형):**
- 중심 노드: 학생이 입력한 문장
- 1차 가지: AI 추출 가치 키워드
- 2차 가지: 연결된 직업군/역량

**동작 흐름:**
1. STEP 2 완료 후 추출된 키워드가 자동으로 전달
2. "마인드맵 만들기" 버튼 클릭
3. AI가 중심-가지형 마인드맵 생성
4. 학생이 색상/배치를 간단히 수정 가능
5. 완성된 마인드맵을 포트폴리오에 저장 또는 공유 보드에 게시

### 3-4. 라이브 디지털 워크시트

| 항목 | 내용 |
|------|------|
| 실시간 협업 보드 | 학생들의 작성 내용이 포스트잇 형태로 실시간 표시 |
| 공개 범위 | 같은 반 학생들만 |
| 멀티미디어 기록 | 텍스트, 사진 업로드, 유튜브 링크 첨부 지원 |
| 반응 기능 | 친구 포스트잇에 좋아요/응원 댓글 달기 |

**동작 흐름:**
1. 수업 시작 시 교사가 '세션 열기' → 고유 세션 코드 생성
2. 학생들이 세션 코드로 입장
3. 각자 작성한 내용이 실시간으로 공유 보드에 반영
4. 수업 종료 시 교사가 '세션 닫기' → 결과 자동 저장

### 3-5. 게이미피케이션 (꿈 포인트 & 레벨)

| 활동 | 포인트 |
|------|--------|
| 문장 입력 및 AI 분석 완료 | +10 꿈P |
| 질문에 대한 답변 작성 | +15 꿈P |
| 진로 한 문장 생성 | +20 꿈P |
| 친구에게 응원 댓글 달기 | +5 꿈P |
| 멀티미디어 자료 첨부 | +10 꿈P |

**레벨 체계:**

| 레벨 | 필요 포인트 | 칭호 |
|------|-----------|------|
| Lv.1 | 0 | 꿈 씨앗 |
| Lv.2 | 50 | 꿈 새싹 |
| Lv.3 | 150 | 꿈 탐험가 |
| Lv.4 | 300 | 꿈 설계사 |
| Lv.5 | 500 | 꿈 크리에이터 |

- 각 탭 내 번호순 정렬

|------|------|

| 수업 | 책 제목, 테마, 참여 시간, STEP별 소요 시간 |

| 소셜 | 좋아요 받은 수, 댓글 받은 수 |

## 6. 사용자 시나리오

### 시나리오 1: 첫 수업 — "나의 가치 발견하기"

- **Who**: 중학교 1학년 김하늘 (자유학기제 진로탐색 수업)
- **When**: 자유학기제 첫 번째 진로 독서 수업 (45분)
- **Where**: 교실, 크롬북 사용
- **Why**: 책 속 인물의 가치관을 통해 자신의 가치관을 탐색하고 싶음
- **How**:
  1. 크롬북에서 꿈틀 접속 → 반 선택 + 번호 입력으로 로그인
  2. 홈 화면에서 "수업 참여하기" 클릭 → 교사가 공유한 세션 코드 입력
  3. STEP 1: 오늘의 책 테마 확인 — "도전과 실패"
  4. STEP 2: 책을 읽으며 인상 깊은 문장 입력 → AI 키워드 분석 결과 확인
  5. STEP 3: 추출된 키워드로 마인드맵 자동 생성 → 색상 수정 후 저장
  6. STEP 4: AI 코치 질문 2개에 대한 답변 작성
  7. STEP 5: '나만의 진로 한 문장' 생성
  8. STEP 6: 공유 보드에서 같은 반 친구들 결과 보기, 좋아요/응원 댓글
  9. STEP 7: 오늘 획득한 꿈 포인트 확인, 레벨업!

### 시나리오 2: 교사 — "수업 운영과 모니터링"

- **Who**: 국어 담당 박선생님 (자유학기제 진로 독서 수업 운영)
- **When**: 수업 준비 및 수업 중
- **Where**: 교실, 교사용 PC/태블릿
- **Why**: 30명 학생의 활동을 실시간으로 파악하고 피드백하고 싶음
- **How**:
  1. 수업 전: 꿈틀에서 '새 세션 만들기' → 책 제목, 테마, 참여 반, STEP별 시간 배분, AI 질문 난이도 설정
  2. 수업 중: 대시보드에서 학생별 작성 현황 실시간 확인
  3. 참여가 저조한 학생에게 개별 알림 또는 직접 격려
  4. 수업 후: 구글 시트에서 반별·번호순 활동 기록 확인, 학부모 상담 자료로 활용

## 7. MVP 범위

### 필수 (MVP v1.0)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 로그인 (반+번호) | 교사 일괄 등록, 학생 반+번호 로그인 | P0 |
| 홈 화면 | 내 레벨 + 최근 활동 + 수업 참여하기 버튼 | P0 |
| 문장 입력 & AI 키워드 분석 | 혼합형 키워드 추출 + 직업군 연결 | P0 |
| AI 질문 생성 | 난이도별 질문 2개 제시 | P0 |
| 진로 한 문장 생성 | 활동 결과를 한 문장으로 정리 | P0 |
| 7단계 가이드 화면 | 단계별 자동 전환 수업 화면 | P0 |
| 실시간 공유 보드 | 같은 반 포스트잇 형태 실시간 공유 | P0 |
| 세션 관리 (교사) | 세션 생성 + 상세 설정 | P0 |
| 반응형 웹 | 크롬북 브라우저 최적화 | P0 |
