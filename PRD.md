# 꿈틀 (Dream-Frame) — PRD

## 1. 서비스 개요
꿈틀은 중학교 1학년 자유학기제 학생들을 위한 AI 진로 독서 플랫폼이다. 기존 종이 워크북의 기록 휘발성 문제를 해결하고, AI 맞춤형 피드백과 실시간 공유 기능을 통해 학생들이 독서를 통해 자신의 진로를 탐색할 수 있도록 돕는다. 1인 1크롬북 수업 환경에서 즉각적으로 사용할 수 있는 웹 기반 도구이다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning) ✅ 상세 완료
- 인증: 교사 일괄 등록 → 학생 반+번호 로그인 (8반 × 30명, 약 240명)
- 화면 흐름: 로그인 → 홈(레벨+최근활동) → 수업 참여하기 → 세션 코드 → 7단계 가이드
- AI 진로 큐레이터: 혼합형 키워드 추출 (6개 카테고리 기본 목록 + AI 추가 제안) + 직업군 연결 (공식 분류 + 학생 친화 설명)
- AI 독서 코치: 난이도 3단계 (쉬움/보통/심화), 각 2개 질문 생성
- AI 비주얼 싱킹: 중심-가지형 마인드맵 자동 생성
- 라이브 디지털 워크시트: 같은 반 공유 보드 + 멀티미디어 기록
- 게이미피케이션: 꿈 포인트 & 5단계 레벨 시스템
- 교사 세션 설정: 책 제목, 테마, 참여 반, STEP별 시간 배분, AI 질문 난이도
- 구글 시트 연동: 반별 탭 + 번호순 정렬 + 전체 활동 데이터 기록
- 에러 처리: 네트워크 장애 임시저장/복구, 부적절 입력 필터링, 세션 이탈/재입장, 시간 초과 자동 전환, 교사 실시간 제어 (STEP 조정/일시정지/연장), 크롬북 배터리 경고

상세는 `planning/PLANNING-PRD.md` 참조.

## 4. 디자인 (Design)
중학생 눈높이에 맞는 친근하고 직관적인 UI. 크롬북 화면 크기에 최적화된 반응형 디자인.
도입(테마 확인) → 독서·키워드 입력 → 진로 문장 생성 → 공유 보드 순서의 화면 흐름.

상세는 `design/` 폴더 참조. [작성 예정]

## 5. CX (Customer Experience)
자유학기제를 맞이한 중학교 1학년 학생. 자기 탐색이 활발하지만 진로에 대한 구체적 경험은 부족한 시기.
수업 도입(5분) → 독서·기록(20분) → 진로 문장 생성(10분) → 공유·피드백(10분)의 45분 수업 여정.

상세는 `cx/` 폴더 참조. [작성 예정]

## 6. 마케팅 (Marketing)
자유학기제 운영 학교 교사 네트워크를 통한 초기 사용자 확보.
교육청 연수, 교사 커뮤니티(인디스쿨 등), 에듀테크 박람회 채널 활용.

상세는 `marketing/` 폴더 참조. [작성 예정]

## 7. 전략 (Strategy)
학교 단위 구독 모델 또는 교육청 예산 기반 도입. 비영리 교육 목적 무료 제공도 검토.
자유학기제 → 전 학년 확대 → 타 교과 연계 독서 활동으로 성장.

상세는 `strategy/` 폴더 참조. [작성 예정]
