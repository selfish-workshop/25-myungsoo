# 꿈틀 (Dream-Frame)

독서를 통해 나를 발견하고, 실시간으로 꿈을 공유하는 AI 진로 독서 플랫폼

## 프로젝트 정보
- **유형**: AI 기반 교육 플랫폼 (EdTech)
- **타겟 사용자**: 중학교 1학년 (자유학기제 대상)
- **사용 환경**: 1인 1크롬북 수업 환경

## 핵심 기능
- AI 진로 큐레이터 & 독서 코치 (키워드 추출, 질문 생성기)
- 라이브 디지털 워크시트 (실시간 협업 보드, 멀티미디어 기록)
- 게이미피케이션 (꿈 포인트 & 레벨 시스템)
- AI 비주얼 싱킹 (키워드 기반 마인드맵 자동 생성)

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
