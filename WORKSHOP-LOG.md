## 3단계: PRD 빌더 실행 — 정보 수집

대화형으로 프로젝트 정보를 수집했습니다.

### 기본 정보
- **프로젝트명**: 꿈틀(Dream-Frame) — 꿈을 담는 틀
- **한 줄 정의**: 독서를 통해 나를 발견하고, 실시간으로 꿈을 공유하는 AI 진로 독서 플랫폼
- **유형**: AI 기반 교육 플랫폼 (EdTech)
- **타겟**: 중학교 1학년 (자유학기제 대상, 자기 탐색이 가장 활발한 시기)
- **사용 환경**: 1인 1크롬북 수업 환경

### 문제 의식
기존 종이 워크북은 기록의 휘발성이 강하고, 친구들과의 실시간 공유나 AI의 맞춤형 피드백을 받기 어려움.

### 핵심 기능

**기능 1: AI 진로 큐레이터 & 독서 코치**
- 키워드 추출: 감명 깊은 문장 입력 → AI가 가치 분석 → 관련 직업군/역량 연결
- 질문 생성기: "이 주인공의 선택이 나의 미래라면?" 같은 맞춤형 질문 제공

**기능 2: 라이브 디지털 워크시트**
- 실시간 협업 보드: 진로 가치관/비주얼 싱킹 결과물을 포스트잇 형태로 실시간 공유
- 멀티미디어 기록: 사진, 유튜브 링크 등 첨부 가능

**기능 3: 게이미피케이션** (추가)
- 활동 완료 시 '꿈 포인트' 획득 및 레벨업 시스템

**기능 4: AI 비주얼 싱킹** (추가)
- 학생이 입력한 키워드를 바탕으로 AI가 마인드맵 이미지 자동 생성

### 수업 활용 시나리오 (45분)
| 단계 | 시간 | 활동 |
|------|------|------|
| 도입 | 5분 | 크롬북으로 앱 접속, 오늘 읽을 책 테마 확인 |
| 전개 1 | 20분 | 독서 + 키워드 입력 → AI 진로 역량 제시 → 생각 기록 |
| 전개 2 | 10분 | 기록 바탕으로 '나만의 진로 한 문장' 생성 |
| 정리 | 10분 | 공유 보드에서 친구 생각 보기, 좋아요/응원 댓글, 교사 대시보드 확인 |
